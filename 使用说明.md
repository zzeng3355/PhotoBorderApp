# Photo Border App 使用说明

## 安装方法

### 方法一：Android Studio编译
1. 下载并安装 [Android Studio](https://developer.android.com/studio)
2. 打开Android Studio，选择 "Open an existing project"
3. 选择 `PhotoBorderApp` 文件夹
4. 等待Gradle同步完成（需要网络）
5. 连接手机或启动模拟器
6. 点击顶部工具栏的绿色三角形按钮（Run）

### 方法二：GitHub Actions下载APK
1. 将代码推送到GitHub仓库
2. 进入仓库的 Actions 页面
3. 找到最新的 workflow run
4. 下载 `app-debug` 或 `app-release` artifact
5. 将APK文件传输到手机安装

## 使用步骤

### 1. 选择图片
- 点击 "Select Images" 按钮
- 从相册中选择一张或多张图片（最多100张）
- 选择后会显示第一张图片的预览

### 2. 选择模板
- 点击 "Template: zzeng_minimal" 按钮
- 从列表中选择边框模板
- 默认模板为zzeng_minimal（极简白底边框）

### 3. 添加边框
- 点击 "Batch Process" 按钮
- 等待处理完成（显示进度条）
- 处理后的图片保存在手机存储的 `Android/data/com.zzeng.photoborder/files/` 目录

### 4. 查看结果
- 处理完成后会显示成功提示
- 可以在文件管理器中找到处理后的图片
- 图片格式为JPEG，质量100（像素无损）

## 增加新模板

### 通过AI对话生成
1. 描述你想要的边框样式，例如：
   - "我要一个黑色背景的胶片风边框"
   - "字体用金色，显示在图片底部"
   - "左侧显示相机型号，右侧显示拍摄参数"

2. 获得JSON配置文件后：
   - 将JSON文件放入手机的 `Android/data/com.zzeng.photoborder/files/templates/` 目录
   - 重启APP即可在模板列表中看到

### 手动创建模板
参考 `assets/templates/zzeng_minimal.json` 的格式，创建自己的模板文件。

## 字体选择

### 选择字体
1. 点击主界面的 **"Font: 微软雅黑"** 按钮
2. 弹出字体选择对话框
3. 从30款字体中选择
4. 点击确认，字体即时生效

## 常见问题

### Q: 为什么处理后的文件比原图大？
A: 因为使用JPEG quality=100保存，保留了最大质量，文件会比原图大约30-50%。

### Q: 支持哪些图片格式？
A: 支持JPEG、PNG等常见格式，输出统一为JPEG。

### Q: 批量处理最多支持多少张？
A: 一次最多处理100张图片。

### Q: 如何更换字体？
A: 点击主界面的 **"Font"** 按钮，从30款字体中选择。
